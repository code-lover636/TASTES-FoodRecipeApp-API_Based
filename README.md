# Tastes

## Description 
Tastes is a food Recipe web app built with Spoonacular API
See hosted website http://coder636.pythonanywhere.com/

## Tools Used
1. Python, Flask for back-end
2. HTML, SCSS, Javascript for front-end
3. Spoonacular API
